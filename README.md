## Android Project: Contact Form App using XML, JAVA & SQLite DataBase
## Features:
1: Pick Image From Gallary or Take real time picture by accessing camera.

2: Take User Inputs.

3: Store Details In SQLite DataBase.

4: Show Details in DropBox.
